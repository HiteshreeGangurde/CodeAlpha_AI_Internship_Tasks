# NLP FAQ Chatbot 🤖

## 📌 Description
This is a Python-based FAQ chatbot that uses Natural Language Processing (NLP) techniques to answer user queries related to Python concepts.
The chatbot uses TF-IDF vectorization and cosine similarity to find the most relevant answer from a predefined dataset.

## 🚀 Features
* NLP-based query matching
* Instant responses
* Simple and interactive GUI using Tkinter
* Handles multiple Python-related questions

## 🛠 Tech Stack
* Python
* Tkinter
* Scikit-learn (TF-IDF, Cosine Similarity)

## ▶️ How to Run
1. Install dependencies:
   pip install scikit-learn

2. Run the file:
   python chatbot.py
