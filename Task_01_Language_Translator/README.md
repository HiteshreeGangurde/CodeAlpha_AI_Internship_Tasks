# 🌐 Language Translator Tool

## 📌 About the Project
This is a simple Python-based Language Translator Web App built using Streamlit and Google Translator API.
It allows users to enter text and translate it into different languages instantly with auto language detection.

## 🚀 Features
* 🌍 Translate text into multiple languages
* ⚡ Fast and lightweight web app
* 🎯 Auto language detection
* 💻 Simple and clean user interface

## 🛠️ Technologies Used
* Python
* Streamlit
* deep-translator

## 📂 Project Structure
language-translator/
│── app.py
│── run_app.bat
│── requirements.txt
│── README.md

## ▶️ How to Run This Project

### Step 1: Clone the repository
git clone https://github.com/your-username/language-translator.git
cd language-translator

### Step 2: Install dependencies
pip install -r requirements.txt
### Step 3: Run the app
streamlit run app.py
OR simply double-click:
run_app.bat

## 🌐 How to Use
1. Enter the text you want to translate
2. Select the target language
3. Click on **Translate**
4. View the translated output instantly

## 📦 Requirements
Make sure you have:
* Python 3.x installed
